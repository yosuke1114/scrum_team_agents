# スプリントプランニング

スプリントプランニングセレモニーを実施します。

## 手順

1. `ceremony_start` type: "planning" でセレモニーを開始
2. READY 状態のタスクを優先度順に一覧表示
3. ユーザーと協議してスプリントゴールを策定
4. スプリントに含めるタスクを選択
5. `sprint_create` でスプリントを作成（ゴールとタスクを指定）
6. `ceremony_end` type: "planning" でセレモニーを終了

## 注意事項

- スプリントゴールは具体的で達成可能なものにする
- WIP 制限を考慮してタスク数を決定する
